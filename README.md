# 🐄 QurbaniHat - Booking Platform

## 📌 Project Overview

QurbaniHat is a modern livestock marketplace where users can explore animals (cows and goats) for Qurbani. Users can view animal details, sort by price, and place bookings after authentication.

## 🌐 Live URL

[View Live Demo](https://altaf-qurbani-hat.netlify.app/)

## ✨ Key Features

### Core Features

- ✅ **Animal Marketplace** - Browse cows and goats with detailed information
- ✅ **Authentication System** - Email/Password and Google Social Login
- ✅ **Booking System** - Book animals after login (form resets on submit)
- ✅ **Price Sorting** - Sort animals by price (Low to High / High to Low)
- ✅ **Private Routes** - Protected pages for authenticated users only

### Challenge Features

1. **My Profile Page** - View user profile with name, email, and photo
2. **Update Profile** - Update user name and profile image URL
3. **Animations** - Smooth animations using Animation css

### Additional Features

- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🎨 Unique Design with Tailwind CSS + daisyUI
- 🔔 Toast Notifications for user actions
- ⚡ Loading States for better UX
- 🚫 404 Not Found Page

## 🛠️ Technologies Used
